# Vector Paint

App de colorir vetorial feito com React, TypeScript, Vite e Tailwind CSS.

## Demo

[Acesse aqui](https://haroldorodsilva.github.io/vector-paint/)

## Desenvolvimento

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desenvolvimento
npm run dev

# Build para producao
npm run build

# Rodar testes
npm run test
```

## Tecnologias

- React 19
- TypeScript
- Vite
- Tailwind CSS 4
- Vitest
